# Logistic Regression
Logistic Regression is a **supervised machine learning algorithm** used for **classification**, mainly binary classification problems (0/1, Yes/No).

It predicts the probability of a class using the sigmoid function and applies a threshold to determine the final output.

Use Cases
- Spam detection  
- Loan approval  
- Disease prediction  
- Customer churn analysis  

 Libraries Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- StandardScaler
